Phase-reversed push-pull

The system consist of three cell populations: ganglion cells, relay cells and cortical cells.

  • The spatial part of the reseptive field for the ganglion cell is a DoG function, while the temporal part is a temporal delta function (no memory).
  • The ganglion cell population is connected to relay cells with a spatiotemporal delta function.
  • The relay cell population is connected to cortical cells with a spatiotemporal delta function.
  • The cortical cell population is connected to relay cells with a temporal delta function, while the spatial part is a Gauss function.

Impulse response functions

\begin{align} \widetilde{W}_{\mathrm{G}_i}(\mathbf{k}, \omega) &= \tilde{f}_{\mathrm{DoG}}(\mathbf{k}) \; \mathcal{F} [ \delta(t) ](\omega) = \tilde{f}_{\mathrm{DoG}}(\mathbf{k}) e^{(-i\omega\; 0)} = \tilde{f}_{\mathrm{DoG}}(\mathbf{k}) \\ \widetilde{W}_{\mathrm{R}}(\mathbf{k}, \omega) &= \frac{ \widetilde{K}_{{\mathrm{R}} \mathrm{G}}\widetilde{W}_{\mathrm{G}} } { 1 - \widetilde{K}_{{\mathrm{R}} \mathrm{C}} \widetilde{K}_{{\mathrm{C}} \mathrm{R}} } = \frac{ \mathcal{F} [ \delta(\mathbf{r})\delta(t) ](\mathbf{k}, \omega)\; \tilde{f}_{\mathrm{DoG}}(\mathbf{k}) } { 1 - \mathcal{F} [ \delta(\mathbf{r})\delta(t) ](\mathbf{k}, \omega)\; \tilde{f}_{\mathrm{Gauss}}(\mathbf{k}) } = \frac{ \tilde{f}_{\mathrm{DoG}}(\mathbf{k}) } { 1 - \tilde{f}_{\mathrm{Gauss}}(\mathbf{k}) } \\ \widetilde{W}_{\mathrm{C}}(\mathbf{k}, \omega) &= \widetilde{K}_{{\mathrm{C}} \mathrm{R}}\widetilde{W}_{\mathrm{R}} = \mathcal{F} [ \delta(\mathbf{r})\delta(t) ](\mathbf{k}, \omega)\; \widetilde{W}_{\mathrm{R}} = \widetilde{W}_{\mathrm{R}} \end{align}
