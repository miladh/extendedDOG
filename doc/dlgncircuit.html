The dLGN Circuit

In LGN, relay cells receive excitatory input from retinal ganglion cells as well as feedforward inhibition from intrageniculate interneurons both via dendro-dendritic structures (triads) and (possibly) axonal output. The interneurons in turn receive excitation from a few retinal ganglion cells. In addition, the relay cells receive inhibitory feedback from the TRN and excitatory feedback from the striate cortex. The intrageniculate interneurons and TRN cells also receive excitatory feedback from cortex. All these feedforward and feedback connections will eventually be included in our present mathematical treatment.

dLGN circuit.

The firing-rate responses of the various cells in the early visual pathway are assumed to be governed by the following set of equations:

\begin{align} R_\mathrm{G}(\mathbf{r},t) &= W_{\mathrm{G}}(\mathbf{r},t) \otimes S(\mathbf{r},t), \\ R_\mathrm{C}(\mathbf{r},t) &= K_{\mathrm{C} \mathrm{R}}(\mathbf{r},t) \otimes R_{\mathrm{R}}(\mathbf{r},t), \\ R_\mathrm{I}(\mathbf{r},t)\; &= \sum_{k} K_{\mathrm{I}\mathrm{G}_k}(\mathbf{r},t) \otimes R_{\mathrm{G}_k}(\mathbf{r},t) \\ &+\sum_{k} K_{\mathrm{I}\mathrm{C}_k}(\mathbf{r},t) \otimes R_{\mathrm{C}_k}(\mathbf{r},t), \\ R_\mathrm{R}(\mathbf{r},t) &= \sum_{k} K_{\mathrm{R}\mathrm{G}_k}(\mathbf{r},t) \otimes R_{\mathrm{G}_k}(\mathbf{r},t) \\ &+\sum_{k} K_{\mathrm{R} \mathrm{I}_k}(\mathbf{r},t) \otimes R_{\mathrm{I}_k}(\mathbf{r},t) \\ &+\sum_{k}K_{\mathrm{R} \mathrm{C}_k}(\mathbf{r},t) \otimes R_{\mathrm{C}_k}(\mathbf{r},t), \end{align}

where the sums run over different populations (e.g. on-centre and off-centre ganglion cell populations) of specific cell types (ganglion cells, interneurons, etc.). Using the results from previous section we can write the impulse response function for various cells:

\begin{align} \widetilde{W}_{\mathrm{C}_i}(\mathbf{k}, \omega) &= \widetilde{K}_{{\mathrm{C}_i} \mathrm{R}}\widetilde{W}_{\mathrm{R}} \\ \widetilde{W}_{\mathrm{I}_i}(\mathbf{k}, \omega)\; &= \sum_j \widetilde{K}_{{\mathrm{I}_i} \mathrm{G}_j}\widetilde{W}_{\mathrm{G}_j} + \sum_j \widetilde{K}_{{\mathrm{I}_i} \mathrm{C}_j} \widetilde{W}_{\mathrm{C}_j} \\ &=\sum_j \widetilde{K}_{{\mathrm{I}_i} \mathrm{G}_j}\widetilde{W}_{\mathrm{G}_j} + \sum_j \widetilde{K}_{{\mathrm{I}_i} \mathrm{C}_j} \widetilde{K}_{\mathrm{C}_j \mathrm{R}} \widetilde{W}_{\mathrm{R}} \\ \widetilde{W}_{\mathrm{R}}(\mathbf{k}, \omega) &= \sum_j \widetilde{K}_{{\mathrm{R}} \mathrm{G}_j}\widetilde{W}_{\mathrm{G}_j} +\sum_j \widetilde{K}_{{\mathrm{R}} \mathrm{I}_j}\widetilde{W}_{\mathrm{I}_j} +\sum_j \widetilde{K}_{{\mathrm{R}} \mathrm{C}_j} \widetilde{W}_{\mathrm{C}_j} \\ &= \sum_j \widetilde{K}_{{\mathrm{R}} \mathrm{G}_j}\widetilde{W}_{\mathrm{G}_j} +\sum_j \widetilde{K}_{{\mathrm{R}} \mathrm{I}_j} \bigg( \sum_k \widetilde{K}_{{\mathrm{I}_j} \mathrm{G}_k}\widetilde{W}_{\mathrm{G}_k} + \sum_k \widetilde{K}_{{\mathrm{I}_j} \mathrm{C}_k} \widetilde{K}_{\mathrm{C}_k \mathrm{R}} \widetilde{W}_{\mathrm{R}} \bigg) +\sum_j \widetilde{K}_{{\mathrm{R}} \mathrm{C}_j} \bigg( \widetilde{K}_{{\mathrm{C}_j} \mathrm{R}}\widetilde{W}_{\mathrm{R}} \bigg) \\ &=\frac{ \displaystyle\sum_j \widetilde{K}_{{\mathrm{R}} \mathrm{G}_j}\widetilde{W}_{\mathrm{G}_j} +\sum_j \widetilde{K}_{{\mathrm{R}} \mathrm{I}_j} \sum_k \widetilde{K}_{{\mathrm{I}_j} \mathrm{G}_k}\widetilde{W}_{\mathrm{G}_k} } { 1 - \displaystyle\sum_j \widetilde{K}_{{\mathrm{R}} \mathrm{I}_j} \sum_k \widetilde{K}_{{\mathrm{I}_j} \mathrm{C}_k} \widetilde{K}_{\mathrm{C}_k \mathrm{R}} - \sum_j \widetilde{K}_{{\mathrm{R}} \mathrm{C}_j} \widetilde{K}_{{\mathrm{C}_j} \mathrm{R}} } \end{align}
